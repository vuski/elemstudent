화면 상단에서 학교 이름으로 검색하거나, 지도를 직접 확대 축소하여 학교를 찾을 수 있습니다.

지도를 확대하면 나타나는 선 그래프는 2008-2023년 각 학교의 학생 수 변화를 보여줍니다.

학교를 클릭하면 추가 정보를 볼 수 있습니다.

'지도 변경' 버튼을 누르면 2008년과 2023년 두 시점의 학생 수를 별도로 볼 수 있습니다. 원의 면적은 학생 수에 비례합니다.
